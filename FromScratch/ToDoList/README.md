# FromScratch — HelloWorld: To‑Do App

Small, vanilla JavaScript To‑Do example demonstrating add/edit/toggle/delete, filters, and persistence via `localStorage`.

How to run
- Open `FromScratch/HelloWorld/index.html` in your browser.

Features
- Add tasks (Enter or Add button)
- Toggle complete, edit inline, delete
- Filter: All / Active / Completed
- Task counter and Clear completed
- Client‑side persistence using localStorage (`fromscratch-hello-todo:v1`)

Accessibility & keyboard shortcuts
- Add: focus input + Enter
- Toggle from list: focus task and press Enter or Space
- Edit: click edit button, Enter to save, Esc to cancel
