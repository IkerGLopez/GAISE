🎵 Guau.eus MP3 URL Extractor

How it works:
This demo shows how to extract the direct MP3 URL from Guau.eus using their public API. Enter a page slug (e.g., from the URL path) and click "Extract MP3 URL" to see the API response.

API Request Details

// API Endpoint Pattern
GET https://guau.eus/api/v1/media/{slug}

// Example Request
const slug = "euskadi-irratia-kantu-kontari-20260220-1505";
const response = await fetch(`https://guau.eus/api/v1/media/${slug}`);
const data = await response.json();
const mp3Url = data.manifests[0].manifestURL;

// Result
console.log(mp3Url);
// Output: "https://cdn.primeran.eus/media/audios/23lufs_..."

Response Structure

Path:
data.manifests[0].manifestURL
Format:
MP3 with 23 LUFS normalization
CDN:
cdn.primeran.eus

Registration required (session auth)
